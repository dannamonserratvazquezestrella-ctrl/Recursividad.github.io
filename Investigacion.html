Introducción

La recursividad es una técnica fundamental en programación donde una función se llama a sí misma para resolver un problema más pequeño hasta llegar a un caso base. Esta metodología permite resolver problemas complejos de manera elegante y estructurada.

Objetivo del Proyecto

El objetivo de este sitio web es investigar, explicar y demostrar mediante ejemplos prácticos el uso de la recursividad en diferentes algoritmos clásicos como el factorial, la serie de Fibonacci, las Torres de Hanói y los fractales.
